## 关键设计 (新增)
- **`Adapter` 模式统一 Exporter**：`IReportFileExporter` 接口 + `PdfSharpExporterAdapter` / `ClosedXmlExporterAdapter` 包装类（避免 dynamic 反射，也避免 generic 约束侵入原有类）；`ReportFileExporter.ExportBatchAsync` 复用 ExportCoreAsync 渲染一次双导出
- **强耦合 PNG 导出用 callback 模式**：MainWindow 在闭包内保存 `oldZoom` + 提供 `renderAt100` / `renderAtCurrent` 两个 `Action` callback
- **`ViewSwitcher.Switch` 显式传 5 个 UIElement + 1 Action callback** + 缺 `using System;` (CS0246 Action 在 net462 必须显式 using)
- **`KeyNudgeCalculator.TryGetDelta` 返回 `ValueTuple<double,double>?` null 表示非方向键**，比 switch/case + bool handled 更紧凑
- **`MarqueeSelector.Select` 返回 `List<(ReportElement, Band)>`**，避免调用点改写签名
- **`ResizeHandleDetector.Detect` 返回手柄编号 0-7 (NW..W), -1 表示未命中**；配合常量 `NoHandle`
