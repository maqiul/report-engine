## 🌐 Web 版本 - Java Spring Boot 后端完成！

### 完成内容
- ✅ Spring Boot 4.1.0 项目骨架
- ✅ Gradle 9.5.1 构建配置
- ✅ RenderController - 渲染预览 API
- ✅ ExportController - PDF/Excel 导出 API
- ✅ iText 7 集成（PDF 生成）
- ✅ Apache POI 集成（Excel 生成）

### API 测试结果
```
POST /api/render/preview  ✅ 正常返回渲染结果
POST /api/export/pdf      ✅ 返回 1148 bytes PDF
POST /api/export/excel    ✅ 返回 3458 bytes Excel
```

### 技术要点
- Spring Boot 4.x 使用 `tools.jackson.databind`（非 `com.fasterxml.jackson`）
- 端口配置为 5000（与 .NET 后端一致）
- CORS 已配置允许 `localhost:5173` 和 `localhost:3000`

### 项目结构
```
web/backend/
├── build.gradle
├── settings.gradle
└── src/main/java/com/reportengine/
    ├── BackendApplication.java
    └── web/
        ├── controller/
        │   ├── RenderController.java
        │   └── ExportController.java
        └── model/
            ├── RenderRequest.java
            └── RenderResponse.java
```

### 启动命令
```bash
cd web/backend
./gradlew bootRun       # 开发运行
./gradlew build -x test # 构建 JAR
java -jar build/libs/backend-0.2.0.jar  # 运行 JAR
```
