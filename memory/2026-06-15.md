### 关键发现（本轮踩坑）
- `MultiUp` 是 `MultiUpConfig?`（单数），不是 `List<MultiUpConfig>`
- `CompositeTemplateResolver` 用 `AddResolver()` 链式添加，不是构造器注入
- `ExpressionEngine.Evaluate(string text, RenderContext context)` — 2 参数
- `Serialize(null)` 不抛异常，返回序列化结果
- 元素类型解析是**大小写敏感**的（必须小写 "text", "image" 等）
- Band 类型解析是**大小写不敏感**的（"Detail" 和 "detail" 都行）
- Band 类型 JSON 用 camelCase：`reportHeader`, `groupHeader` 等
- `{{UNKNOWN_VAR}}` 返回去括号后的变量名 `"UNKNOWN_VAR"`
- `{{PAGE}}`, `{{TOTAL_PAGES}}`, `{{ROW_NUMBER}}`, `{{REPORT_DATE}}`, `{{NOW}}` 是有效系统变量
- `ITemplateResolver` 接口有 `Exists(string)` 方法
- `RenderContext.CurrentPage` 默认 **1**（不是 0）
- `RenderContext.TotalPages` 默认 **1**（不是 0）
- `RenderContext.PageWidth` 默认 **210**，`PageHeight` 默认 **297**
- `TemplateNotFoundException` 有 `TemplateRef` 和 `SearchPath` 属性
- `FileSystemTemplateResolver` 有缓存机制（同一路径返回同一对象）
- `FileSystemTemplateResolver` 有 `Save()` 和 `ListTemplates()` 方法
- `ReportElement.Id` 自动生成 GUID，每个实例唯一
- `TextElement.BoxType` 是计算属性：SysVar > Summary > Field > Static
- `MultiUpConfig.Count` 是计算属性 = Rows × Columns
- `Band.SubBands` 默认 **null**（不是空列表）
- `Band.DataSource` 默认 **null**（不是空字符串）
- `ReportTemplate.CreatedAt` 默认 **DateTime.Now**（不是 MinValue）
- `ReportTemplate` 没有 `Name` 和 `Styles` 属性
- `ReportTemplate.Parameters` 类型是 `List<TemplateParam>`（不是 `ParameterDef`）
- `TemplateParam` 没有 `Prompt` 和 `Required` 属性
- `DataSourceDef.Type` 默认 **"json"**（不是 null）
- `DataSourceDef.Fields` 默认空列表
- `FieldDef.Type` 默认 **"string"**
- `FieldDef.ToString()` 包含 name、type 和 format
- `ConditionalFormatRule` 只有 4 个属性（Expression/BackgroundColor/FontColor/Bold），**没有 Italic 和 Underline**
- `MultiColumnConfig` 默认 ColumnCount=2, ColumnSpacing=5, Direction="Horizontal"
- `TableElement` 默认 RowCount=3, ColCount=3, BorderWidth=0.3, BorderColor="#000000"
- `TableCell` 默认 Alignment=Center
